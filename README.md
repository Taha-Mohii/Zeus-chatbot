# 🤖 Zeus AI Chatbot

Zeus is a command-line AI chatbot built using the Groq API.  
It supports conversational memory, context-aware responses, and acts as a personalized assistant.

---

## 🚀 Features

- 🧠 Conversation memory (remembers previous messages)
- 🤖 Context-aware AI responses
- ⚡ Powered by Groq API (LLaMA 3)
- 🛡 Error handling for API failures
- 👤 Personalized assistant (Zeus)

---

## 🛠 Tech Stack

- Python
- Groq API
- dotenv
